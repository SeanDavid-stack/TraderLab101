# Code of Conduct

## Our Pledge

TraderLab 101 is a tool built by a trader for traders. We're here to help each other get better at the craft. Be respectful, be helpful, be honest.

## Standards

- Be constructive with feedback — "this broke when I did X" is useful, "this sucks" is not
- Respect that everyone is at a different stage in their trading journey
- No financial advice — this tool is for journaling and self-improvement, not trade recommendations
- Keep discussions focused on the tool and methodology

## Scope

This applies to GitHub issues, pull requests, and any discussions related to this project.

## Enforcement

Disruptive behavior will result in comments being removed and repeat offenders being blocked from the repository.

## Contact

Open an issue on the repository or reach out via the Traders Lab Discord.
